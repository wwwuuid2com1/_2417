<h3>源码简介：</h3>
<p>有时候经常遇到wifi密码忘记的情况，导致连接不上去
有这个软件就可以查看你电脑使用过的wifi账号密码
双击打开软件即可，等待片刻自动加载本机WiFi信息。
在列表中右击菜单可以复制WiFi密码或名称，也可以备份全部信息，方便重装系统造成WiFi密码丢失的困扰。<p>
